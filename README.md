# @z-os/ui v4.2.0

zOS UI Components - Window, dock, menubar. Built on @hanzo/ui.

## Install
```bash
npm install @z-os/ui
```

## Components
- ZWindow - macOS-style window
- ZDock - Dock with magnification
- Icons - All macOS-style icons

## Hooks
- useWindowManager
- useDesktopSettings
- useKeyboardShortcuts
- useIsMobile
